# Directory Map: app

- `build/` - Build output directory specifically for the app module.
- `src/` - Source code for the app module, structured into main, test, and androidTest source sets.
- `.gitignore` - Specifies intentionally untracked files in the app module for Git to ignore.
- `build.gradle.kts` - Build script for the app module, defining plugins, dependencies, and Android-specific configurations.
- `proguard-rules.pro` - Rules for code shrinking, obfuscation, and optimization using ProGuard/R8.
- `FILE_MAP.md` - This file mapping the current directory structure.
